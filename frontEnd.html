Front-end

"프론트엔드(front-end)"는 사용자가 웹사이트나 앱을 사용할 때
직접 눈으로 보고, 클릭하거나, 스크롤하면서 사용하는 화면을 만드는 부분입니다.
예를 들어, 웹사이트의 버튼, 메뉴, 이미지, 텍스트 같은 사용자 인터페이스(UI) 요소가 포함됩니다.

프론트엔드 개발은 'HTML', 'CSS', 'JavaScript'와 같은 기술을 사용합니다.
HTML로 뼈대를 만들고, CSS로 디자인을 꾸미고, JavaScript로 움직이는 기능을 추가합니다.

사용자가 입력한 데이터를 처리하거나 서버를 통신하는 백엔드(back-end)와 상호작용합니다.
